# AI Assignment 1: Data Analysis and Machine Learning

## 📌 Objective
To analyze a dataset and build machine learning models to predict pulse rate.

## 📊 Dataset
The dataset includes features such as diet, pulse, time, and activity type.

## ⚙️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor

## 📈 Results
Random Forest performed better with:
- Lower RMSE
- Higher R² Score

## 💡 Insights
- Pulse increases with physical activity
- Time duration impacts pulse significantly
- Random Forest captures complex patterns better
